    float

    一个浮点数。

    一个实数的有限精度表示。Typst 使用 64 位来存储浮点数。 在需要使用浮点数的任何地方，你也可以传递一个 integer。

    Example

    #3.14 \
#1e4 \
#(10 / 4)
