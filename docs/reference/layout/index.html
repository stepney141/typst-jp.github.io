    • 布局

    layout

    以不同方式在页面上排列元素。

    通过组合布局函数，您可以创建复杂且自动的布局。

    定义

    • align在水平和垂直方向上对齐内容。
    • alignment在沿轴线的何处[对齐]($align)某物。
    • angle描述旋转的角度。
    • block一个块级容器。
    • box一个内联级容器，可调整内容的大小。
    • colbreak强制列中断。
    • columns将区域分割为多个大小相等的列。
    • direction内容可以布局的四个方向。
    • fraction定义布局中剩余空间的分布方式。
    • grid将内容排列成网格。
    • h在段落中插入水平间距。
    • hide隐藏内容而不影响布局。
    • layout提供对当前外部容器（如果没有，则是页面的）大小的访问。
    • length一个大小或距离，可能使用上下文单位表示。
    • measure测量内容的布局大小。
    • move移动内容而不影响布局。
    • pad在内容周围添加间距。
    • page将其子元素布局到一个或多个页面上。
    • pagebreak手动分页。
    • place在绝对位置放置内容。
    • ratio整体的比率。
    • relative相对于某个已知长度的长度。
    • repeat将内容重复到可用空间。
    • rotate旋转内容而不影响布局。
    • scale缩放内容而不影响布局。
    • stack水平或垂直地排列内容和间距。
    • v在块流中插入垂直间距。
