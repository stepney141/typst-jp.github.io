    cite 元素
    元素函数可以通过 set 和 show 规则进行样式自定义。

    从参考文献中引用一项作品。

    在开始引用之前，您需要在文档的某个位置添加一个参考文献。

    示例

    这早在海盗时代就已经被注意到了。
@arrgh

多个来源说 ...
@arrgh @netwok。

您还可以显式调用 `cite`。
#cite(<arrgh>)

#bibliography("works.bib")

    语法

    此函数间接具有专用语法。可以使用引用来引用参考文献中的作品。然后，标签对应于引用键。

    参数
    参数是函数的输入，它们在函数名称后面的括号中传入。

    cite(
    label, supplement: nonecontent, form: nonestr, style: autostr,
    ) -> content

    key
    label
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    标识在参考文献中的要引用的条目的引用键。

    查看示例 
    // 都是相同的
@netwok \
#cite(<netwok>) \
#cite(label("netwok"))

    supplement
    none content
    可用 Set 规则
    可以使用 Set 规则，为函数后续所有的使用设置默认参数。

    引文的补充，例如页码或章节编号。

    在引用语法中，可以在方括号中添加补充：

    默认：none

    查看示例 
    这已经被证明了。 @distress[p.~7]

#bibliography("works.bib")

    form
    none str
    可用 Set 规则
    可以使用 Set 规则，为函数后续所有的使用设置默认参数。

    要生成的引文类型。不同的形式在不同情境中很有用：标准引文适用于句子末尾的来源，而“散文”引文更适合在文本流中包含。

    如果设置为none，则引用的作品将包含在参考文献中，但不会显示任何内容。

    • "normal"

      以活动样式的标准方式显示。

    • "prose"

      生成适用于句子中包含的引文。

    • "full"

      模拟参考文献条目，提供有关引用作品的完整信息。

    • "author"

      仅显示引用作品的作者。

    • "year"

      仅显示引用作品的年份。

    默认："normal"

    查看示例 
    #cite(<netwok>, form: "prose")
显示海盗生活对人类心灵的巨大影响。

    style
    auto str
